# Véu de Dados  
**Auditoria algorítmica ex ante para justiça distributiva**
## Objetivo  
Ferramenta open-source para auditoria de vieses em algoritmos de crédito e políticas públicas automatizadas.  
Baseada no conceito de "Véu de Dados", implementa princípios de accountability algorítmico e simulação distributiva ex ante.  
Embora o primeiro módulo seja focado no crédito, o sistema foi desenhado para escalar a outros domínios (emprego, justiça criminal, educação, saúde, apoios sociais).
## Funcionalidades  
- **Anonimização crítica** de dados sensíveis, preservando proxies auditáveis.  
- **Simulação distributiva ex ante** com thresholds éticos configuráveis.  
- **Testes de gémeos** para deteção de vieses ocultos.  
- **Relatórios visuais** com dashboards de justiça algorítmica.
## Justificação Ética e Técnica  
O projeto baseia-se no conceito de "Véu de Dados", uma proposta original inspirada na teoria da justiça de John Rawls, adaptada à era digital.  
Este conceito permite simular ex ante os efeitos de decisões algorítmicas, identificar padrões de exclusão e garantir accountability algorítmico.  
Trata-se de um mecanismo que vai além da transparência: é uma forma concreta de submeter o código à justiça, antes de este afetar vidas reais.  
O Véu de Dados é o “método”; o accountability algorítmico é o “resultado esperado”.
